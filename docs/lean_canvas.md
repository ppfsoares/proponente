# Lean Canvas - Soma Alerta

O Lean Canvas abaixo sumariza o modelo de negócio da plataforma, focando em agilidade e entrega de valor para o ecossistema cultural.

| **Problema** | **Solução** | **Proposta de Valor Única** | **Vantagem Injusta** | **Segmentos de Clientes** |
| :--- | :--- | :--- | :--- | :--- |
| 1. Dispersão de informações em Diários Oficiais e sites governamentais. | 1. Indexação automatizada e centralizada de editais. | O "Curador Digital" da cultura nordestina: Democratizando o acesso ao fomento através de IA e linguagem simples. | Algoritmo de Match Semântico treinado especificamente para a burocracia cultural brasileira e foco regional (NE). | 1. Artistas Independentes e Coletivos Periféricos. |
| 2. Linguagem técnica e burocrática de difícil compreensão. | 2. Tradução de editais complexos para linguagem simples via IA. | | | 2. Produtores e Captadores de Recursos. |
| 3. Exclusão de proponentes com baixa capacidade técnica (especialmente no NE). | 3. Sistema de alertas proativos (WhatsApp/Email) baseados no perfil. | | | 3. Gestores Públicos/Privados (B2G/B2B). |

| :--- | :--- | :--- |
